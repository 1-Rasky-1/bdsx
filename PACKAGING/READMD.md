## Build Typescript (Watch Mode)
You need to install [NodeJS](https://nodejs.org/en/) before use Typescript

* Build with VSCode
1. Open `bdsx/` with VSCode
2. Ctrl + Shift + B
3. Select `tsc: watch`

* Build with Command Line
1. Open `bdsx/` with Prompt
2. run `npm watch`
